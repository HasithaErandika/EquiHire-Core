# EquiHire Frontend Documentation

This directory provides documentation specific to the EquiHire frontend application.

---

## Documentation Categories

1.  **[Frontend Design Guidelines](./frontend-design.md)**
    Overview of the UI/UX principles, design system tokens, and component architecture used in the project.

2.  **[Identity and Authentication Lifecycle](./identity-lifecycle.md)**
    Technical details on the integration with WSO2 Asgardeo, including organization provisioning, recruiter login flows, and the candidate magic-link invitation system.

3.  **[API Integration Reference](./api-endpoints.md)**
    Detailed specification of the communication protocols between the React client and the Ballerina Gateway, including CV upload and assessment triggers.

4.  **[System Architecture Overview](./introduction.md)**
    High-level perspective on how the frontend interacts with the internal microservices and external AI managed services.

5.  **[Global Project Documentation](../../doc/README.md)**
    Link to the core system documentation repository.

## Technical Stack Reference

- **Core:** React, TypeScript, Vite.
- **Styling:** Tailwind CSS, Framer Motion.
- **Components:** Radix UI primitives.
